# Music Listening Habits Analysis

This project analyzes people's music listening habits using statistical techniques and Python programming.

## Features
- Survey Data Analysis
- Pie Chart Visualization
- Statistical Calculations
- Correlation Analysis
- Python-based Data Processing

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Forms

## Statistical Methods
- Mean
- Median
- Mode
- Variance
- Standard Deviation
- Skewness
- Kurtosis

## Output
- Pie Charts
- Survey Analysis
- Statistical Interpretation
